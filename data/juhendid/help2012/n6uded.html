Nõuded

EELex on brauserisõltumatu, välja arvatud skeemieditor, morfoloogia lisamine ja automaatne klaviatuurivahetus. Sõnastiku kujul väljatrükkide tegemiseks on vajalik MS Word.

Veebilehitseja häälestus

Veebilehitseja peab olema häälestatud järgmiselt.

1. Eesti Keele Instituudi serverid peavad olema kantud usaldusväärsete kohtade (Trusted sites) nimekirja (järgnev on Internet Exploreri näitel):
Tools/Tööriistad ->
Internet Options/Interneti suvandid ->
Security/Turvalisus sakk (kaart) ->
Trusted sites/Usaldusväärsed kohad ->
nupp Sites/Saidid ->
kirjutada lahtrisse "eki.ee",
eemaldada linnuke "Require server verification…
Küsi serveri kontrollimist… (https:) … kastikeselt ->
nupp Add/Lisa ->
nupp Close/Sule.
2. Usaldusväärsete kohtade (Trusted sites) tsooni turvalisussätteid tuleb muuta järgmiselt.
Samal, Security/Turvalisus sakil (kui tsooniks on valitud Trusted Sites/Usaldusväärsed kohad) nupp Custom level…/Kohandatud tase…:
Initialize and script ActiveX controls not marked as safe for scripting/Lähtesta ja skripti ActiveX-i juhtelemendid, mis pole tähistatud skriptimiseks turvalistena -> Enable/Luba.
Access data sources across domains/Luba juurdepääs muude domeenide andmeallikatele -> Enable/Luba.
Use Pop-up Blocker/Kasuta hüpikublokki -> Disable/Keela.
